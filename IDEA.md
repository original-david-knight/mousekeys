# Mouse Keys Idea
A tool that runs on my system which allows me to
move my mouse and click using just the keyboard. 
It is triggered by a specific key combination. After
the tool is triggered, the user can enter a
coordinate to move the mouse to, or they can enter a
command to click or double click.  the coordinates 
are given with letters of the alphabet.  the coordinate
grid is overlayed on the screen when the app is waiting
for a coordinate. after a coordinate is chosen, the
mouse moves there and a sub-cell grid opens and user can
optionally further refine the position.  escape exits.
